# memo_project
- 메모 기능을 사용하고 싶어서 만든 프로젝트
- 로그인 기반, 메모 저장 기능, 사진 저장 기능, 삭제,업데이트 기능
- Spring을 통한 프로젝트 경험을 위한 프로젝트
